WiFi Signature Server

How can I add my device?

There are two ways:

  1. Add device.json to WiFi Signature Communitry Repository.
  2. Ask for maintainer account.

Where can I find my WiFi Signature?

You need OpenWrt with full wpad installation.

In the shell you can use

ubus call hostapd.[INTERFACE] get_clients

to findout the device signatures.

For example:

ubus call hostapd.wlan0 get_clients
