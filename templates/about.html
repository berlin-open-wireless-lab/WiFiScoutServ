About

This is a project that tries to collect WiFi Signatures.
It is based on the paper Passive Taxonomy of Wifi Clients using MLME Frame Contents
There is already a WiFi Signature GitHub-Repositry.
We try to find a way to offer the information in a more user friendly way. You can just query our API.
You can easily add devices through the Django web interface.
You can easily export those devices to a json format.
